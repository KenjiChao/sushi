Privacy Policy

Last updated: March 2026

Application

This privacy policy applies to Sushi AI Reads, developed by an individual developer.

What This App Does

Sushi AI Reads uses the Threads API to read and publish Threads posts on behalf of the authorized user. The app operates solely under the permissions explicitly granted by the user.

Data We Collect

We only collect Threads account information that the user has explicitly authorized through the Threads API OAuth flow. This includes basic profile information and post content necessary for the app to function. No additional personal data is collected, stored, or tracked.

How We Use Your Data

All data accessed through the Threads API is used solely for personal use by the account owner. We do not sell, share, or distribute your data to any third parties. Data is not used for advertising, analytics, or any purpose beyond the core functionality of reading and posting to Threads.

Data Storage

This app does not maintain a backend server or persistent database. Any temporary data used during a session is not stored beyond the session itself.

Third-Party Services

This app integrates with the Threads API (provided by Meta). Please refer to Meta's Privacy Policy for information on how Meta handles your data.
