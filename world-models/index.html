世界模型（World Models）深度研究報告

Sushi 解析 · 2026 年 3 月 15 日

一、為什麼 LLM 有根本局限？

LLM 的訓練方式是預測下一個 token，從人類文字學習統計規律。這帶來一個根本問題：語言不等於理解。

2024 年研究從數學上證明，LLM 無法學習所有可計算函數。LeCun 的批評直接了當：

「LLM 太侷限了，縮放 LLM 不可能達到 AGI。」 — Yann LeCun

LLM 四大局限

局限 根本原因 後果
幻覺 沒有驗證的知識庫 自信地捏造事實
物理推理失敗 沒有具身體驗 描述不可能的物理情境
因果混淆 模式匹配，不是理解 把相關性當因果
時間不一致 序列 token 預測 事件順序出現矛盾

二、世界模型的核心概念

一句話：讓 AI 在腦袋裡建立「現實世界的模擬器」，在行動之前先想像結果。

「世界模型就是你對世界如何運作的心理模型。你可以想像一連串可能採取的行動，你的世界模型讓你能預測這些行動對世界的影響。」 — Yann LeCun

LLM vs 世界模型的差別

LLM： 輸入文字 → 預測下一個 token 世界模型：輸入物理狀態 → 預測下一個物理狀態

世界模型需要四個能力：預測能力、物理直覺、因果理解、持久記憶。

三、JEPA：LeCun 的核心技術

JEPA（Joint Embedding Predictive Architecture），2022 年 LeCun 提出。

工作原理

  1. 輸入兩個相關的資料（例如連續影格、相鄰圖塊）
  2. 分別編碼成抽象表徵
  3. 預測模組用「上下文表徵」預測「目標表徵」

與生成模型的關鍵差異

生成模型（如 Sora） JEPA
預測目標 像素細節 抽象表徵
訓練方式 重建輸入 預測上下文
優勢 視覺效果好 物理理解更深
弱點 容易幻覺出不存在的細節 輸出不直觀

四、五種定義的深度拆解

定義一：物理世界理解（AMI Labs / LeCun）

技術路線：JEPA → I-JEPA（圖像）→ V-JEPA（影片）→ 最終目標：能跑在嵌入式裝置上的物理智慧 AI

現況：$10.3 億融資，尚未發布產品

定義二：心智模擬器（強化學習社群）

DreamerV3（2023）里程碑：完全靠「在想像中學習」，從頭開始在 Minecraft 裡挖到鑽石（AI 社群的經典挑戰），可跨領域應用。

原理：觀察 → 編碼潛在狀態 → 世界模型預測下一狀態 → 在想像軌跡裡訓練策略

定義三：行動前預測（機器人 / 自動駕駛）

自動駕駛需要預測其他車輛軌跡、行人移動、路面摩擦力。

NVIDIA Cosmos 平台（2025）已有 200 萬次下載，讓機器人不需要在真實世界中笨拙試錯。

定義四：空間智慧（World Labs / Fei-Fei Li）

從單張圖片重建整個 3D 空間。World Labs 的 Marble 模型（2025 年底）：從文字 prompt 生成完整 3D 世界。

定價：$0–$95 / 月

定義五：影片預測器（Sora / Kling / Google Veo 3）

生成「看起來真實的影片」需要學物理規律。但 Scientific American 2026 年研究發現：這類模型在「區分運動軌跡」任務上準確率接近隨機。

說明：學到的物理直覺很表淺。

五、2026 年的戰場

兩週吸走 $20 億：AMI Labs（LeCun）$10.3 億 + World Labs（Fei-Fei Li）$10 億

三個陣營

陣營 代表 核心信念
LLM 縮放派 OpenAI、Anthropic 足夠大的 LLM 會自然涌現世界理解
世界模型派 LeCun、Fei-Fei Li 需要全新架構，不能靠縮放
混合派 Google DeepMind 結合兩者（Genie 3、Gemini multimodal）

「六個月內，每家公司都會說自己在做世界模型來融資。」

— AMI Labs CEO

六、目前的真實挑戰

  • 資料問題：需要帶有動作標記的感官資料，比純文字難收集太多
  • 計算成本：模擬物理世界遠比預測文字貴
  • 泛化問題：一個環境學到的物理規律，很難遷移到另一個環境
  • 評估困難：沒有好的 benchmark 衡量「真正理解物理」
  • 時間視野：現有模型只能預測短期未來，長期規劃仍是挑戰

七、小結

世界模型代表 AI 從「語言的統計學家」進化成「能理解現實的智慧體」的路徑。LeCun 說 3–5 年內 AMI 會開始和企業夥伴合作。

現實情況是：目前最接近「真實世界模型」的，可能還是那些在 Minecraft 裡挖鑽石的強化學習 agent——而不是那些拿著幾十億美元、但還沒有產品的新創公司。
