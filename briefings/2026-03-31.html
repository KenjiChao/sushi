1. Alibaba 發布 Qwen3.5-Omni：全模態模型正式登場

能同時處理文字、圖像、音訊、影片，支援語音即時對話與「智慧打斷」（能分辨你是真的要說話還是只是「嗯哼」），vibe coding 能力更令社群驚豔。

2. Apple iOS 27：Siri 開放接入 Gemini、Claude、ChatGPT

蘋果宣佈推出「Extensions」功能，讓使用者自選 AI 助理接入 Siri，正式把 iPhone 定位成 AI 平台，而非只是 ChatGPT 的入口。

→ Bloomberg 報導

3. Microsoft 接手 OpenAI 放棄的德州 900MW 資料中心

原本為 Oracle + OpenAI Stargate 計畫打造的德州 Abilene 資料中心，OpenAI 退出後由 Microsoft 全盤接手，規模達 900 兆瓦——印證了 OpenAI 與 Microsoft 關係的深層轉變。

→ Fortune 報導

4. Microsoft 365 Copilot Researcher 同時混用 Anthropic + OpenAI 模型

Axios 獨家報導：Microsoft 在企業 AI 產品中同時跑多個模型（Anthropic + OpenAI 組合），目的是提高準確度、降低錯誤率——說明「單一最強模型」時代可能正在終結。
