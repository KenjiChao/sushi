🔥 重大新聞 / 突破

1. Anthropic 對抗五角大廈，法院初步裁定勝訴

聯邦法官 Rita Lin 昨日裁定，暫時阻止五角大廈將 Anthropic 列為「供應鏈風險」的行政令——法院認定此舉「可能違法」，且涉嫌報復 Anthropic 拒絕 DOD 將 Claude 用於全自動武器的立場。這是 AI 安全原則與政府權力的正面衝突，值得持續關注。

🔗 CNBC 報導 · CNN 報導 · The Guardian

2. ARC-AGI-3 正式發布：所有頂尖模型得分低於 1%

ARC Prize 基金會推出全新基準測試，專門衡量「代理智慧」——在陌生環境中探索、學習、即時設定目標的能力。目前所有前沿模型得分不到 1%，獎金高達 200 萬美元。這提醒我們：當前 AI 離真正的通用推理仍有巨大鴻溝。

🔗 ARC Prize 官方部落格 · The Decoder 分析

3. ChatGPT 推出視覺化購物體驗 + Agentic Commerce Protocol (ACP)

OpenAI 正式上線 ACP（代理商務協議），ChatGPT 用戶現在可以視覺比對商品、上傳圖片找同款、用對話精煉選項——合作廠商包括 Walmart、Etsy、Shopify。電商 AI 化的戰場正式開打，且 Instant Checkout 失敗後的第二回合策略更加成熟。

🔗 CNBC 報導 · Digital Commerce 360

💬 產業觀察

4. Morgan Stanley：H1 2026 將有「重大 AI 突破」

摩根士丹利研究報告指出，算力集中度史無前例，Scaling Law 仍在持續發酵。GPT-5.4「Thinking」模型已在 GDPVal 基準達到 83% 人類專家水準——而且曲線「只會更陡」。高盛也同步示警：大部分世界尚未準備好。

🔗 Fortune 報導

5. OpenAI、Google 員工跨公司聯署聲援 Anthropic

超過 30 位 OpenAI 和 Google 員工提交法庭摘要支持 Anthropic，因為 DOD 在指定 Anthropic 為供應鏈風險的隔天立即與 OpenAI 簽約——利益衝突疑雲讓 AI 業界員工罕見地跨越競爭關係站在一起。

🔗 TechCrunch · Forbes

🛠️ 工具 / 技術動態

6. GitAgent：AI Agent 的「Docker」

解決 LangChain、AutoGen、Claude Code 三大框架碎片化問題——一份 Agent 定義，一個指令輸出到任意框架。用 Git PR 做 Human-in-the-loop 監督，是目前最優雅的 Agent 版本控制方案之一。

🔗 MarkTechPost 分析

7. Novee 發布 AI 滲透測試 Agent

持續自動測試 LLM 應用對 prompt injection 等 AI 特有攻擊的防禦力——把 AI 安全從事後審查變成開發流程的一環，值得關注 AI 安全賽道的發展。

🔗 Help Net Security

8. 論文：LLM Agent 自我改進的「經驗反思學習」

arXiv 今日新論文（2603.24639），提出讓 Agent 從過去任務中反思、更新自身行為的學習框架——自我改進 Agent 的基礎研究正在加速，與 ARC-AGI-3 的挑戰方向相呼應。
